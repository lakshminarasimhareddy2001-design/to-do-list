## вҡҷпёҸ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/task-manager.git
```

2. Navigate to project folder:

```bash
cd task-manager
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```
