
A simple and responsive **Task Manager (Todo App)** built using **React + Redux Toolkit + TypeScript**.
It allows users to add, toggle, and delete tasks with a clean UI.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* ⌨️ Add task using **Enter key**
* 📱 Fully responsive (mobile-friendly)
* 🎨 Styled with custom CSS

---

## 🛠️ Tech Stack

* **React** (Functional Components)
* **TypeScript**
* **Redux Toolkit**
* **CSS (Flexbox + Media Queries)**

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/lakshminarasimhareddy2001-design/to-do-list.git
```

2. Navigate to project folder:

```bash
cd to-do-list
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```

---

## 🎯 Usage

* Type a task in the input field
* Press **Enter** or click **Add**
* Click on a task to mark it as completed
* Click **Delete** to remove a task
